This is an alpha version of a reservation system for hostels, hotels and other kind of accomodations.

== Description ==

A simple WordPress Plugin to generate reservations and generally organize room reservations for small hotels and hostels.

== Installation ==

This section describes how to install the plugin and get it working.

1. Install the plugin:
    1. Upload the `joeee-booking.zip` file at wp-admin > Plugins > Add New menu
    1. Or, via SFTP, upload the `joeee-booking` directory to the `/wp-content/plugins/` directory
1. Activate the plugin:
    1. Visit wp-admin > Plugins
    1. Find the `Jö's booking` plugin in the list
    1. Click its "Activate" link
1. Use the plugin:
    1. Change the settings in the settings panel of your WP Installation
    2. Use the Jö's booking section in the WordPress admin.
